Video/Content APIs

GEOMETRY DASH RANDOM VIDEO API

Retrieve random gameplay videos from Geometry Dash sourced from TikTok. Explore exciting gameplays shared by the community!

Try it: /api/gd

RANDOM EDIT VIDEO API

Access a random selection of video edits, including motion graphics and content-based edits from popular apps like Alight Motion.

Try it: /api/edit

TIKTOK DOWNLOADER API

Download TikTok videos directly using the video URL for offline access.

Try it: /api/tikdl?url=your video link from TikTok

FACEBOOK VIDEO DOWNLOADER API

Download Facebook or Facebook Reels videos using their URLs with ease.

Try it: /api/fbdl?url=your video link from Facebook

YOUTUBE DOWNLOADER API

Use this API to download YouTube videos through their URL for offline viewing.

Try it: /api/yt?url=your video link from YouTube
Social Media APIs

TIKTOK SEARCH API

Search TikTok videos based on hashtags or specific keywords. Find popular content in seconds!

Try it: /api/tiktok/searchvideo?keywords=

TIKTOK STALK API

Explore TikTok users' profiles by their unique IDs. Get details about their videos and activities.

Try it: /api/tikstalk?unique_id=

INSTAGRAM STALK API

Access Instagram profiles by username to view photos, videos, and other publicly available information.

Try it: /api/instagram/stalk?ig=

PINTEREST API

Search Pinterest images based on a title or keyword to discover creative content.

Try it: /api/pin?title=wallpaper&count=15
AI-Related APIs

GPT API

Interact with GPT-3.5 for text-based responses. You can ask anything and get intelligent answers.

Try it: /api/gpt?prompt=hello

AI API

A simple AI for answering questions. Use it to ask anything and receive responses instantly.

Try it: /api/ai?query=hi

GLOBAL GPT API

Access a global version of GPT-3.5 for more advanced interactions and responses.

Try it: /api/globalgpt?content=hello

LLAMA AI API

Meta's Llama AI to assist with complex queries via FastAPI in Python.

Try it: /meta?prompt=hello
